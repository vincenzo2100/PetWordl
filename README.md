Instrukcja
1. Pobierz klucz API Anthropic.
2. Wklej go do appsetting.json w miejscu "YOUR-API-KEY".
3. Przejdź do głównego katalogu aplikacji.
4. Użyj polecenia "docker compose up" lub "docker compose up --build".
5. Applikacja dostępna jest przy użyciu przeglądarki pod adresem http://localhost:5000/ .

Uwagi!
Nie mogłem przetestować logiki działania agentów ponieważ nie mam darmowego dostępu do api.
Baza danych MySQL została stworzona na stronie db4free, przedstawione dane są pewnego rodzaju mockupem. Nie umieściłbym nigdy kluczy API ani danych logowania na publiczny repozytorium.
